# find_maze_bfs

# BFS를 이용해 최단거리 미로 탐색을 해결하는 시뮬레이션

1. 랜덤 맵 : srand의 랜덤 시드와 rand함수를 이용해 랜덤으로 벽을 생성

2. BFS : 매번 현재 위치에서 4방향중에 가장 목적지와 가까운 곳으로 이동

3. opencv : Mat, rectangle, putText 를 활용해 시뮬레이션 시각화

---

# 사용법

시뮬레이션이 끝날때마다 새로운 맵으로 테스트하고싶으면 아무키나 누르고

종료하고 싶으면 q를 누르면 된다
